# Global Search & Token Paging

Global all-notes search is the main scenario path that demonstrates
`TokenPagedComposition` beyond the simpler fixed-page notes list.

## What It Proves

- forward-only token paging
- search results independent of the current notebook
- the same conceptual flow across all four language flavors
