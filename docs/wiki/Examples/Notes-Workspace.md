# Notes Workspace

Notes Workspace is the flagship VMx example portfolio: one scenario, four
idiomatic hosts, and one shared VM contract.

![Examples VM Layer Map](../../assets/diagrams/examples-vm-layer.png)

Support links: [HTML](../../assets/diagrams/examples-vm-layer.html),
[SVG](../../assets/diagrams/examples-vm-layer.svg),
[PNG](../../assets/diagrams/examples-vm-layer.png)

## What It Exercises

- notebook tree projection
- notes-list selection through `CompositeVM.current`
- strict `FormVM` editing and validation
- derived status and enablement
- dialogs, notifications, and theme state
- mode switching with `DiscriminatorVM`
