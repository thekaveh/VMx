# Builders, Collections & Tree Utilities

Use this area when the question is construction ergonomics, observable
collections, or tree traversal.

## Three Groups

- immutable fluent builders and additive create helpers
- observable collections and paging primitives
- traversal helpers such as walk, find, and expanded-tree walks

## Practical Rules

- builders are immutable and reusable
- required fields validate at build time
- collection batching and VM container batching are separate scopes
- tree utilities are pure reads
