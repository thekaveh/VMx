# FormVM

Use `FormVM<TM>` when users edit a working copy, then approve or deny and
revert to the last snapshot.

![Forms Dialogs And Notifications Flow](../../../../assets/diagrams/forms-dialogs-notifications.png)

Support links: [HTML](../../../../assets/diagrams/forms-dialogs-notifications.html),
[SVG](../../../../assets/diagrams/forms-dialogs-notifications.svg),
[PNG](../../../../assets/diagrams/forms-dialogs-notifications.png)

## What It Owns

- live model and snapshot
- dirty and validation state
- approve and deny commands
- fire-and-forget approve error surface

## Common Uses

- editors
- settings forms
- CRUD dialogs
