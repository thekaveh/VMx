# Hierarchical Family

Use `HierarchicalVM` when the recursive VM tree is itself the right model.

![Hierarchical Family Map](../../assets/diagrams/hierarchical-family.png)

## Best Fit

- explorer-style trees
- recursive outline structures
- expand/collapse navigation over child subtrees

## Important Note

The current Notes Workspace portfolio represents notebook structure through flat
`ComponentVM`-based adapters that preserve tree messaging rather than direct
`HierarchicalVM` subclasses.
