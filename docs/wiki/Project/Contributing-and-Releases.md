# Contributing & Releases

For non-trivial work, start in the repo guides and keep the spec discipline
rules visible.

## High-Level Flow

- open an issue for non-trivial work
- branch from `main`
- run the relevant flavor checks locally
- start behavior changes in `spec/` and add the required ADR
- keep shared behavior aligned across supported flavors
